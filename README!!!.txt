======= Docs Front =======

- Url index barang ( menampilkan barang yang dapat dilihat user )
http://localhost/majo/front/barang

- Login 
http://localhost/majo/front/user/

- check login
http://localhost/majo/front/user/login

- index barang pada admin
http://localhost/majo/front/user/barang

- form tambah barang
http://localhost/majo/front/user/formtambahbarang

- form edit barang (di klik pada nama barang yang ingin dirubah )
http://localhost/majo/front/user/formeditbarang/{nama_barang}

- button delete 
http://localhost/majo/front/user/deletebarang/{nama_barang}

- logout
http://localhost/majo/front/user/logout
